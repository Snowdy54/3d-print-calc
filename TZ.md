# Техническое задание: 3D-Print Analytics

## 1. Цель проекта
Создание системы для автоматизации расчета себестоимости 3D-печати. Сервис позволяет учитывать стоимость расходных материалов в валюте (USD), амортизацию оборудования и рассчитывать рекомендуемую цену для клиента на основе заданных коэффициентов.

## 2. Роли пользователей
* **Администратор:** Управление парком принтеров, каталогом пластика, настройка глобального коэффициента наценки через админ-панель.
* **Менеджер (Пользователь):** Ввод параметров печати (вес, время), просмотр истории заказов и аналитики по обороту материалов.

## 3. Модели данных
* **Printer (Принтер):** Название, стоимость часа работы.
* **Filament (Пластик):** Название, цена за катушку в USD, вес катушки.
* **PrintOrder (Заказ):** Связь с принтером и пластиком, вес модели, время печати, итоговая себестоимость (total_cost).
* **StudioSettings:** Глобальные настройки (коэффициент наценки).

## 4. Ключевой функционал
* **Калькулятор:** Расчет себестоимости с учетом текущего курса USD через внешний API.
* **Аналитика:** Визуализация выручки по принтерам и расхода пластика (Plotly).
* **Бизнес-отчет:** Таблица оборота по типам материалов с использованием агрегации данных (Count, Sum).

## 5. Внешние интеграции / Аналитика
* **API:** ExchangeRate-API для получения актуального курса USD/RUB.
* **Библиотеки:** Plotly для генерации интерактивных круговых диаграмм.
