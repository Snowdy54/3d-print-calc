# 3D-Print Analytics MVP
Аналитическая система для студий 3D-печати. Позволяет мгновенно рассчитывать прибыльность заказов и отслеживать популярность материалов.

## Стек технологий
* **Python 3.10+ / Django 4.2**
* **Plotly** (интерактивная аналитика)
* **Bootstrap 5** (адаптивный UI)
* **ExchangeRate API** (курсы валют)

## Функционал
* Автоматический расчет себестоимости с конвертацией валют.
* Мониторинг оборота по типам пластика.
* Визуализация нагрузки на оборудование.

## Запуск локально
1. Клонировать репозиторий.
2. Создать venv: `python -m venv venv`.
3. Установить зависимости: `pip install -r requirements.txt`.
4. Выполнить миграции: `python manage.py migrate`.
5. Запустить: `python manage.py runserver`.
