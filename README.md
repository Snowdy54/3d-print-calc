# 3D Print Calculator

Система поддержки принятия решений для владельцев студий 3D-печати.

## Ключевые возможности:
- **Авто-расчет:** Учитывает износ сопла, потребление тока и вес пластика.
- **Мультивалютность:** Цены материалов пересчитываются по живому курсу USD через API.
- **Бизнес-аналитика:** Наглядные графики Plotly показывают, какое оборудование приносит больше всего оборота.

## Стек технологий:
- Python 3.x, Django 4.x
- Plotly (Visuals)
- Bootstrap 5 (UI/UX)
- Requests (API Integration)

## Как запустить:
1. `pip install -r requirements.txt`
2. `python manage.py migrate`
3. `python manage.py runserver`
